# LQB-Chanllenge-2026
蓝桥杯 2026 备赛仓库 | 刷题 + 真题 + 模板 + 题解

## 目录结构
- 01-Basic：基础算法题（模拟、枚举、数组、字符串）
- 02-Algorithm：算法专题（DP、搜索、贪心、数论、图论）
- 03-Exam：历年真题
  - Provincial：省赛真题
    - 01-Fill：结果填空题
    - 02-Code：编程题
  - National：国赛真题
    - 01-Fill：结果填空题
    - 02-Code：编程题
- 04-Template：竞赛常用代码模板

## 文件命名规范
### 填空题
年份-赛事-题号.md
例：2025-省赛-填空01.md

### 编程题
年份-赛事-题号.cpp
例：2025-省赛-编程03.cpp

## 代码规范
每题开头标注：题目来源、思路、时间/空间复杂度
