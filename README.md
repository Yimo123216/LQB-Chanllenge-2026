# LQB-Chanllenge-2026
蓝桥杯2026备赛仓库 | 适配结果填空+编程题，真题/模板/刷题全归档

## 目录说明
- 01-Basic：基础算法题（模拟、枚举、数组、字符串），一题一文件
- 02-Algorithm：算法专题（DP、DFS/BFS、贪心、数论、图论），专项突破
- 03-Exam：历年真题（按赛事+题型双维度归档）
  - Provincial：省赛真题
    - 01-Fill：省赛结果填空题（存储答案/解题过程）
    - 02-Code：省赛编程题（C/C++代码+思路+复杂度）
  - National：国赛真题
    - 01-Fill：国赛结果填空题
    - 02-Code：国赛编程题
- 04-Template：蓝桥杯竞赛通用算法模板（直接复用）

## 题型上传规范
### 结果填空题（01-Fill）
- 文件格式：`.md` （可直接复制到Word）
- 内容结构：题目编号 → 题目描述 → 答案 → 关键推导/思路
- 命名示例：`2025省赛_Fill_03.md`（第3道填空题）

### 编程题（02-Code）
- 文件格式：`.cpp` （C/C++竞赛标准）
- 代码开头注释：题目编号 → 题型（填空/编程） → 解题方法 → 时间/空间复杂度
- 命名示例：`2024省赛_Code_07.cpp`（第7道编程题）
